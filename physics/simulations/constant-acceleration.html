Simulation Lab: Constant Acceleration (1D)

This is not a step-by-step lesson. It’s a playground: change \(u\), \(a\), and time, then see how motion and graphs behave instantly.

Mode: Parameter sweep
Graphs: x(t), v(t)
Goal: build intuition

What to try

Exploration
Use these mini-experiments like a lab notebook. Don’t rush—change one parameter at a time.
  • Zero acceleration: set \(a=0\). What happens to \(v(t)\)? What does \(x(t)\) look like?
  • Reverse direction: set \(u>0\) and \(a<0\). Does the object stop? When does \(v=0\)?
  • Gravity feel: set \(a=-9.8\). If \(u>0\), what’s the “top of motion” time?
  • Curvature test: increase \(T\). How does the bend of \(x(t)\) change with bigger \(|a|\)?
  • Same final velocity: try different pairs \((u,a)\) that give the same \(v\) at the same \(t\). What changes?
